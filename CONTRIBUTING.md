# Contributing

Gracias por contribuir a CopyOpsSuite.

## Flujo sugerido

1. Crea un branch desde `main`.
2. Mantn cambios enfocados y con commits claros.
3. Verifica compilacion local (`dotnet build`).
4. Abre un PR con descripcion clara.

## Estilo

- Codigo claro y comentarios minimos.
- Evitar cambios masivos no relacionados.
- Respetar el estilo existente en XAML y C#.
