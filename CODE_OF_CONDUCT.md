# Code of Conduct

Este proyecto sigue un codigo de conducta basado en el Contributor Covenant.

## Nuestro compromiso

Nos comprometemos a mantener un entorno respetuoso y libre de acoso para todos.

## Comportamientos esperados

- Respeto y profesionalismo
- Colaboracion constructiva
- Empatia con los demas

## Reportes

Para reportar incidentes, contacta al mantenedor del repositorio.
